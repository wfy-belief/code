# 求交错序列前N项和
本题要求编写程序，计算交错序列 1-2/3+3/5-4/7+5/9-6/11+... 的前N项之和。

### 输入格式:

输入在一行中给出一个正整数N。

### 输出格式:

在一行中输出部分和的值，结果保留三位小数。

### 输入样例:
```in
5
```

### 输出样例:
```out
0.917
```
### 思路与视频教程
**<font color='#e59572'>慢慢完善</font>**

### 代码
```python
# 分子 1 2 3 = i + 1
# 分母 1 3 5 = 2 * i + 1
# 符号 = -1 ** i
n = int(input())
ans = 0
for i in range(n):
    ans += ((-1) ** i) * (i + 1) / (2 * i + 1)
print(f'{ans:.3f}')

```
