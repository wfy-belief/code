# 字符串逆序
输入一个字符串，对该字符串进行逆序，输出逆序后的字符串。 

### 输入格式：

输入在一行中给出一个不超过80个字符长度的、以回车结束的非空字符串。 

### 输出格式：

在一行中输出逆序后的字符串。 

### 输入样例：
```in
Hello World!
```

### 输出样例：
```out
!dlroW olleH
```

**<font color='#e59572'>慢慢完善</font>**

### 代码
```python
s = input()
print(s[::-1])
```
