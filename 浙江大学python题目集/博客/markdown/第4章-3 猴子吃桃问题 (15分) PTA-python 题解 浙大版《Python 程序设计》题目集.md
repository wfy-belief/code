# 猴子吃桃问题
一只猴子第一天摘下若干个桃子，当即吃了一半，还不过瘾，又多吃了一个；第二天早上又将剩下的桃子吃掉一半，又多吃了一个。以后每天早上都吃了前一天剩下的一半加一个。到第$$N$$天早上想再吃时，见只剩下一个桃子了。问：第一天共摘了多少个桃子？

### 输入格式:

输入在一行中给出正整数$$N$$（$$1<N\le 10$$）。

### 输出格式:

在一行中输出第一天共摘了多少个桃子。

### 输入样例:
```in
3
```

### 输出样例:
```out
10
```
### 思路与视频教程
**<font color='#e59572'>慢慢完善</font>**

### 代码
```python
ans = 1
n = int(input())
for _ in range(n - 1):
    # 3 1 + 1  = ans
    ans = (ans + 1) * 2
print(ans)

```
