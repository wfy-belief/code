# 生成3的乘方表
输入一个非负整数$$n$$，生成一张3的乘方表，输出$$3^0$$~$$3^n$$的值。可调用幂函数计算3的乘方。

### 输入格式:

输入在一行中给出一个非负整数$$n$$。

### 输出格式:

按照幂的递增顺序输出$$n+1$$行，每行格式为“pow(3,i) = 3的i次幂的值”。题目保证输出数据不超过长整型整数的范围。

### 输入样例:
```in
3
```

### 输出样例:
```out
pow(3,0) = 1
pow(3,1) = 3
pow(3,2) = 9
pow(3,3) = 27
```
### 思路与视频教程
**<font color='#e59572'>慢慢完善</font>**

### 代码
```python
n = int(input())
for i in range(n + 1):
    print(f'pow(3,{i}) = {3 ** i}')

```
