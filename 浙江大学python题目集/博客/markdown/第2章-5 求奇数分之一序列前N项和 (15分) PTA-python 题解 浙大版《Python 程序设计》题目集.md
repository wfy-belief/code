# 求奇数分之一序列前N项和
本题要求编写程序，计算序列 1 + 1/3 + 1/5 + ... 的前N项之和。

### 输入格式:

输入在一行中给出一个正整数N。

### 输出格式:

 在一行中按照“sum = S”的格式输出部分和的值S，精确到小数点后6位。题目保证计算结果不超过双精度范围。

### 输入样例:
```in
23
```

### 输出样例:
```out
sum = 2.549541
```
### 思路与视频教程
**<font color='#e59572'>慢慢完善</font>**

### 代码
```python
# an = 2 * n - 1 or 2 * n + 1
n = int(input())
ans = 0
for i in range(n):
    ans += 1 / (2 * i + 1)
print(f'sum = {ans:.6f}')

```
