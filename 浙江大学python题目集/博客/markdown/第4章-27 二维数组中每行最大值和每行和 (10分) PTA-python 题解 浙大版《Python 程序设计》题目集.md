在一行中输入9个小于100的整数，其间各以一个空格间隔

### 输出格式:

输出3行3列的二维数组，并在每行后面分别输出每行最大值和每行元素的和，每个数据输出占4列。

### 输入样例:
```in

```

### 输出样例:
```out

   1   4   5   5  10
```
